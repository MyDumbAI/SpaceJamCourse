Module 7: Image Maps

Introduction to HTML — Rebuilding the Space Jam Website

Goal: Build the Jam Central section hub page. The navigation is a single image with four clickable regions mapped to different links — an image map.

What is an Image Map?

An image map lets you define multiple clickable regions within a single image, each linking to a different destination. Instead of using separate button images, you draw invisible hotspot shapes on top of one image.

The Jam Central hub page uses one large image (m-central.jpg) with four rectangular hotspots mapped to Production Notes, Photos, Filmmakers, and the Theatrical Trailer.

The map and area Tags

An image map requires two things: a <map> element that defines the clickable areas, and a usemap attribute on the <img> that connects the image to the map.

The <map> tag takes a name attribute. Each clickable region inside it is an <area> tag with:

  • shape — the shape of the hotspot: rect (rectangle), circle, or poly (polygon)
  • coords — the pixel coordinates defining the shape
  • href — where to go when that area is clicked
  • target — same as on an <a> tag

Coordinates

For a rectangle, coords are: left, top, right, bottom — the x and y of the top-left corner, then the x and y of the bottom-right corner. All values are in pixels measured from the top-left of the image.

<area shape="rect" coords="33,92,178,136" href="prodnotes.html">

For a circle, coords are: center-x, center-y, radius.

For a polygon, coords are a sequence of x,y pairs for each vertex.

To find the correct coordinates, open the image in any image editor and hover over the corners of each region you want to make clickable. Most editors show the current pixel position in the status bar.

Connecting Image to Map

The usemap attribute on the <img> tag links the image to its map by name, prefixed with a hash:

<img src="assets/m-central.jpg" width=438 height=301 usemap="#map" border=0>

<map name="map">
  <area shape="rect" coords="33,92,178,136" href="prodnotes.html" target="_top">
  <area shape="rect" coords="244,111,416,152" href="photos.html" target="_top">
  <area shape="rect" coords="104,138,229,181" href="filmmakers.html" target="_top">
  <area shape="rect" coords="230,155,334,197" href="trailer.html" target="_top">
</map>

Assets

  • m-central.jpg — the Jam Central navigation image (438×301)
  • bg_stars.gif — star field background tile

Steps

Step 1 — Create jamcentral.html

Create a new file. Set the title to "Jam Central" and apply the Space Jam color scheme to the body: bgcolor="#000000", background="assets/bg_stars.gif", text="#ff0000", link="#ff4c4c", vlink="#ff4c4c", alink="#ff4c4c".

Step 2 — Add the image with usemap

Inside the body, add a <br>, then open a <blockquote> and a <center>. Place the m-central.jpg image with usemap="#map" and border=0.

Step 3 — Define the map

Immediately after the <img> tag (still inside the center/blockquote), add the <map> element with four <area> tags using the coordinates and links above. Each area uses target="_top".

Step 4 — Add the description text and footer

After the map, close </center> and add the introductory paragraph using <font size="+2">:

You've made it: Jam Central Station, the central depository for all things Space Jam. From the best seats in the house, you can peruse the production notes, find out about the filmmakers, check out the theatrical trailer, and look at a bunch of photos from the film.

Close the blockquote, then add a centered copyright footer.

Step 5 — Test the image map

Open the page in a browser. Hover over the image — you should see the cursor change to a pointer over the four hotspot areas. The labels in the image correspond to the four mapped regions: Production Notes, Photos, Filmmakers, and Trailer.

The linked pages (prodnotesframes.html, photosframes.html, etc.) do not exist yet. The links are correct — the pages will be built in later modules.
