# Introduction to HTML: Rebuilding the Space Jam Website

A project-based course where each module introduces new HTML concepts by building real pages from the 1996 Space Jam website. By the end, students will have reconstructed the complete site from scratch.

---

## Module 1: The Skeleton
**Exercise:** Build a blank Space Jam home page — just a black page with a title.
**Topics:** HTML document structure (`<html>`, `<head>`, `<body>`, `<title>`), the role of each tag, saving and opening files in a browser, HTML comments.

---

## Module 2: Text & Typography
**Exercise:** Recreate a character bio page (e.g., Bugs Bunny from The Lineup).
**Topics:** Headings (`<h1>`–`<h6>`), paragraphs, line breaks (`<br>`), `<center>`, `<font>` (size, color, face), `<b>`, `<i>`, `<nobr>`, HTML entities (`&copy;`, `&amp;`).

---

## Module 3: Colors & Backgrounds
**Exercise:** Apply the Space Jam color scheme to the home page — black background, star field, red text.
**Topics:** `bgcolor`, `background` (image), `text`, `link`, `vlink`, `alink` attributes on `<body>`, hexadecimal color values.

---

## Module 4: Links
**Exercise:** Add working navigation links to the bio page and connect it to a second page.
**Topics:** `<a href>`, relative vs. absolute paths, `target` attribute (including `_blank`), linking to external URLs, the `border=0` convention on linked images.

---

## Module 5: Images
**Exercise:** Add the Space Jam logo and section-button images to the home page.
**Topics:** `<img>` tag, `src`, `alt`, `width`, `height`, `border`, `align` (including `absmiddle`), image file formats (GIF vs. JPG), organizing assets in folders.

---

## Module 6: Tables
**Exercise:** Recreate the home page's orbital navigation layout — section buttons positioned around the central logo using a table.
**Topics:** `<table>`, `<tr>`, `<td>`, `<th>`, `border`, `cellpadding`, `cellspacing`, `width`, `height`, `align`, `valign`, `colspan`, `rowspan`, using tables for visual layout.

---

## Module 7: Image Maps
**Exercise:** Build the Jam Central section hub with a clickable image map for navigation.
**Topics:** `<map>`, `<area>`, `shape` (rect, circle, poly), `coords`, `href`, `alt`, linking a map to an image with `usemap`.

---

## Module 8: Frames
**Exercise:** Reconstruct the two-frame layout used by every section — a persistent nav frame alongside a content frame.
**Topics:** `<frameset>`, `<frame>`, `rows` and `cols` attributes, `name`, `src`, `scrolling`, `noresize`, `<noframes>`, targeting frames with `target`.

---

## Module 9: Embedding Media
**Exercise:** Build the Planet B-Ball QTVR viewer page and the Lunar Tunes audio download page.
**Topics:** `<embed>` tag, `src`, `width`, `height`, `autoplay`, linking downloadable files (`.wav`, `.aif`, `.zip`, `.hqx`), organizing media assets, graceful fallback for unsupported formats.

---

## Module 10: Final Project — Assemble the Site
**Exercise:** Wire together all completed sections into a fully navigable site matching the original.
**Topics:** Site-wide file organization, consistent relative paths across nested directories, the sitemap page, verifying every link and image works locally, comparing the finished result to the original.
