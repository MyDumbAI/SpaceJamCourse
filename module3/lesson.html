Module 3: Colors & Backgrounds

Introduction to HTML — Rebuilding the Space Jam Website

Goal: Apply the Space Jam color scheme to the home page skeleton from Module 1 — a black background tiled with a star field GIF, red body text, and a centered Space Jam logo image.

Color on the body Tag

In 1996-era HTML, colors for the whole page are set as attributes on the <body> tag. There is no CSS involved — these attributes control everything at once.

Attribute What it controls
bgcolor Background color of the page
background An image file that tiles across the background
text Color of all body text
link Color of unvisited links
vlink Color of visited links
alink Color of a link while being clicked

Hex Color Codes

Colors are written as a # followed by six hexadecimal digits representing red, green, and blue values (#RRGGBB). Some examples from the Space Jam site:

  • #000000 — black (the page background)
  • #ff0000 — red (body text on the home page)
  • #ff4c4c — a softer red (used for links)
  • #feff89 — yellow (used on the Bugs bio page)
  • #ffffff — white

Background Images

When you provide a background image, the browser tiles it (repeats it) across the entire page. The Space Jam home page uses a small GIF of stars that tiles to fill the black background seamlessly.

<body background="assets/bg_stars.gif">
bgcolor and background together: Setting both is good practice. The bgcolor shows instantly while the image loads, preventing a white flash. The Space Jam page sets both to black/stars.

Images with img

You have seen <img> in earlier modules. Here you will use it to place the Space Jam logo on the page. The key attributes:

  • src — path to the image file
  • width and height — dimensions in pixels (always include these; they prevent layout jumping while images load)
  • alt — text description shown if the image cannot load
  • border=0 — removes the blue border browsers add around images that are inside links
<img src="assets/p-jamlogo.gif" height=165 width=272 alt="Space Jam" border=0>

Assets

The following files are in the assets/ folder for this module:

  • bg_stars.gif — small tileable star field (used as the page background)
  • p-jamlogo.gif — the Space Jam logo

Steps

Step 1 — Start from the Module 1 skeleton

Create a new file called index.html. Paste in the skeleton from Module 1:

<!-- Copyright 1996 Warner Bros. Online -->
<html>

<head>

<title>Space Jam</title>

</head>

<body>

<h1>Space Jam</h1>

</body>

</html>

Step 2 — Add the color scheme to body

Replace the plain <body> tag with one that sets the background color, background image, text color, and link colors:

<body bgcolor="#000000" background="assets/bg_stars.gif" text="#ff0000" link="#ff4c4c" vlink="#ff4c4c" alink="#ff4c4c">

Save and refresh. The page should now be black with a tiled star field and the "Space Jam" heading in red.

Step 3 — Replace the heading with the logo image

Remove the <h1> tag and replace it with a centered logo image:

<center>
<img src="assets/p-jamlogo.gif" height=165 width=272 alt="Space Jam" border=0>
</center>

Save and refresh. You should see the Space Jam logo centered on the star-field background.

Step 4 — Add the copyright footer

Add the copyright notice at the bottom, inside <center> so it is centered, using <font> for sizing:

<br>
<br>

<center>
<font size="-1">SPACE JAM, characters, names, and all related<br>
indicia are trademarks of Warner Bros. &copy; 1996</font><br>
</center>

Save and refresh. Your page now has the full Space Jam look: black starfield, red text, centered logo, and a copyright footer.
