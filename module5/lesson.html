Module 5: Images

Introduction to HTML — Rebuilding the Space Jam Website

Goal: Add all eleven section-button images to the Space Jam home page. Along the way, learn the full set of <img> attributes, the difference between GIF and JPG formats, and how to organize image assets in folders.

img Attributes in Full

You have used <img> since Module 2. Here are all the attributes it supports, as used throughout the Space Jam site:

  • src — path to the image file. Required.
  • alt — text description of the image. Required for accessibility.
  • width and height — dimensions in pixels. Always include these — they prevent layout reflow as images load.
  • border — thickness of the border drawn around the image. Almost always set to 0.
  • align — controls how the image interacts with surrounding content. left and right float the image to that side and let text wrap around it. absmiddle vertically aligns an inline image to the middle of adjacent text.
  • hspace — adds horizontal space (in pixels) to both the left and right of the image.
  • vspace — adds vertical space above and below.

GIF vs. JPG

The Space Jam site uses two image formats. Understanding when to use each is part of working with images.

GIF (Graphics Interchange Format) is best for images with flat areas of solid color, logos, icons, and line art. GIFs support transparency and simple animation. Most of the navigation buttons and title graphics in the Space Jam site are GIFs.

JPG (Joint Photographic Experts Group) is best for photographs and images with gradients or complex color gradations. JPG compresses photographic content much more efficiently than GIF. The character photos (like c-bugs.jpg) are JPGs.

Organizing Assets in Folders

The Space Jam site stores its root-level images in an img/ folder next to the home page. Section-specific images live in an img/ folder inside each section's folder. In this module, all images are in the assets/ folder — the path in your src attribute just needs to match where the file actually is.

<img src="assets/p-jamlogo.gif" width=272 height=165 alt="Space Jam" border=0>
