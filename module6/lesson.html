Module 6: Tables

Introduction to HTML — Rebuilding the Space Jam Website

Goal: Recreate the Space Jam home page's orbital navigation layout — the eleven section buttons positioned around the central logo — using an HTML table.

How Tables Work

An HTML table is made of three nested tags:

  • <table> — the outermost container
  • <tr> — a table row (inside <table>)
  • <td> — a table data cell (inside <tr>)

Every cell sits at the intersection of a row and a column. Columns are implied by how many <td> elements you put in a row — there is no explicit column tag.

Table Attributes

These attributes appear on <table>:

  • border — thickness of the visible border. Set to 0 to hide the grid lines (the Space Jam site uses border=0 for all layout tables).
  • cellpadding — space in pixels between a cell's content and its edges.
  • cellspacing — space in pixels between cells.
  • width — total width of the table, in pixels or as a percentage.

Cell Attributes

These attributes appear on <td>:

  • align — horizontal alignment of the cell's content: left, center, or right.
  • valign — vertical alignment: top, middle, or bottom.
  • width — width of the cell in pixels.
  • colspan — makes a cell span across multiple columns.
  • rowspan — makes a cell span across multiple rows.

Tables for Layout

In 1996, CSS layout did not exist in any practical sense. Tables were the only reliable way to position elements precisely on a page. The Space Jam home page uses a table with width=500 and border=0 to position its section buttons in the circular arrangement around the logo. The logo itself sits in a cell with colspan=3 and rowspan=2 so it occupies a large central area while the buttons surround it.

When using tables for layout, always set border=0, cellpadding=0, and cellspacing=0 to start. Add spacing back only where you need it.

Steps

Create a new index.html starting from your Module 5 completed file. The logo and copyright footer are already in place. You will replace the flat row of section buttons with a table layout.

Step 3 — Build the navigation table

Replace the deleted section with a centered table of width=500 and border=0. The table has four rows arranged as follows:

  • Row 2: Press Box (2-col span, right-aligned) | Jam Central (center) | Planet B-Ball (center, top) | Lunar Tunes (center, bottom)

  • Row 4 (continues row 3 logo span): Junior Jam (center, bottom) | Studio Store (center, top, 2-row span)
  • Row 4: empty cell | Souvenirs (center, top) | Site Map (center, bottom) | Behind the Jam (center, middle)

Refer to completed/index.html for the exact table structure with all colspan, rowspan, align, and valign values.
