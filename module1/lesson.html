Module 1: The Skeleton

Introduction to HTML — Rebuilding the Space Jam Website

Goal: Create the bare skeleton of the Space Jam home page — a valid HTML file with a title, visible in a browser, containing a single heading. No styling yet.

What is HTML?

HTML (HyperText Markup Language) is the language used to structure content on the web. An HTML file is a plain text file that a browser reads and renders as a page. Every web page you have ever visited — including the original 1996 Space Jam site — starts with this same basic structure.

HTML uses tags to mark up content. Most tags come in pairs: an opening tag and a closing tag. The closing tag has a forward slash.

<tagname>content goes here</tagname>

The Four Required Tags

Every HTML page needs these four tags:

  • <html> — Wraps the entire document. Tells the browser "this is HTML."
  • <head> — Contains information about the page (not visible on screen).
  • <title> — Sets the page title shown in the browser tab. Goes inside <head>.
  • <body> — Contains everything visible on the page.

HTML Comments

Comments let you leave notes in your code. The browser ignores them completely. The original Space Jam source begins with a copyright comment.

<!-- This is a comment. The browser ignores it. -->

Headings

HTML has six heading levels, from <h1> (largest) to <h6> (smallest). Use headings to mark titles and section names. Your page needs at least one heading so there is something visible to see.

<h1>Space Jam</h1>

Steps

Step 1 — Create the file

Open a plain text editor (Notepad on Windows, TextEdit on Mac — make sure it is in plain text mode, not rich text). Create a new file and save it as index.html.

The .html extension tells the browser and your operating system that this is a web page.

Step 2 — Add the document structure

Copy and paste the following into your file:

<html>

<head>

<title>Space Jam</title>

</head>

<body>

</body>

</html>

Save the file, then open it in your browser. You should see a blank page with Space Jam in the browser tab.

Step 3 — Add a copyright comment

Comments are invisible to the user but useful for notes and credits. The real Space Jam page opens with one. Add it as the very first line, before <html>:

<!-- Copyright 1996 Warner Bros. Online -->
<html>

<head>

<title>Space Jam</title>

</head>

<body>

</body>

</html>

Step 4 — Add a visible heading

Put a heading inside <body> so the page has something to show:

<!-- Copyright 1996 Warner Bros. Online -->
<html>

<head>

<title>Space Jam</title>

</head>

<body>

<h1>Space Jam</h1>

</body>

</html>

Save and refresh. You should now see "Space Jam" as a large heading on a white page. This is your skeleton.

What You Built

Your finished file should match completed/index.html. It is minimal on purpose — future modules will layer on colors, images, links, and layout. Every page in the Space Jam site, no matter how complex, is built on exactly this same skeleton.
